# UST Visualizer | UST 可视化工具

利用deepseek制作的**UST可视化工具**

使用python语言，pyinstaller打包

## 安装依赖库（exe版本可省略）：
`pip install pygame pillow`

## 使用教程：

### 格式转换：

由于程序仅支持ust文件，所以需要先将你的工程文件**转换为ust文件**

这里推荐：https://utaformatix.tk/

转换格式选"Ust - UTAU工程"

注意勾选"转换音高参数"，否则音高曲线将无法正常使用

### 基本使用：

启动程序后会出现命令窗口，**请勿关闭**

点击左上角"选择UST文件"，选择你刚刚转换的UST文件

点击"选择输出文件夹"，选择你想要输出到的文件夹（请务必自己建好，程序不会自行生成文件夹）

接下来点击左下角的"生成序列帧"即可开始渲染，可在命令窗口查看进度

生成完毕后，请将序列帧导入**视频处理软件**中转换为视频

### 参数/其他：
采取从UI布局从左向右，自上而下的顺序进行介绍

**1. 选择字体文件**：你可以选择自己准备的字体文件来作为歌词的字体

**2. 保存/加载配置**：你可以将自己所做的个性化调整储存在json，以便下次继续使用

**3. 基本设置**：

-分辨率设置：调整分辨率，最高支持4k

-其他设置：调整帧率(最高60帧)，调整歌词字体大小

**4. 动画设置**：

-滚动速度：即下落速度，值越大，单个音符越长，流动越快

-判定线位置：左右调整判定线

-淡出淡入效果：在开头/结尾使序列逐渐显现/消失

-纵向位置调整：上下调整整体的位置

**5. 样式设置**：

-音符样式：调整音符的上下厚度和阴影

-歌词位置：调整歌词的垂直偏移

-音高曲线样式：调整宽度、阴影和平滑度等

**6. 颜色设置**：更改背景、音符和歌词等的颜色
